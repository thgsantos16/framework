PASTAS E CONTEÚDOS

- estilos - Essa pasta recebe os estilos em cascata e respectivos assets
- documentos - pasta que armazena todos os tipos de documentos enviados à plataforma
- img - nessa pasta estão alocadas todas as imagens do sistema
- includes - pasta que armazena os arquivos estruturais do sistema
- node_modules - pasta para inserção de módulos Node.js


/*******************************************************************/

ARQUIVOS

- index.php - Índice do Sistema, página para Construção do Sistema.
- login.php - Autenticação no Sistema
- logout.php - Limpeza das Sessões, que permite uma nova Autenticação no Sistema
- readme.txt - Arquivo com as Instruções Essenciais do SIstema
- web.config - Configurações do Servidor HTTP, como permissões, tratamento de URL e reescrita.
